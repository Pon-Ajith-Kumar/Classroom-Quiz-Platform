# Offline Classroom Quiz Platform

A real-time quiz system for 9 teams and 1 teacher, designed to run fully offline over a local hotspot.

## Run

```bash
npm install
node server.js
```

Server URLs:

- Team: `http://<laptop-ip>:3000`
- Teacher: `http://<laptop-ip>:3000/teacher`

For local test:

- `http://localhost:3000`
- `http://localhost:3000/teacher`
